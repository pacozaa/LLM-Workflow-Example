# Backend

A NestJS backend service for the LLM Workflow Example application.

## Description

This is the main backend service that handles API requests for the application.

## Port

This service runs on port **3001**.

## Installation

```bash
npm install
```

## Running the app

```bash
# development
npm run dev

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
